# docker-compose-microservices

Example project using:

* Docker 19.03.x
* Docker Compose 1.25.x


## Installation

Install dependencies using:

    docker-compose build


## Usage

Run the server:

    docker-compose up

View the frontend at:

    http://localhost:8080


## Directory structure

    /src                      --> Frontend source files
